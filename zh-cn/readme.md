## 功能

- 储存、管理和预览 HTML, [MHTML](https://en.wikipedia.org/wiki/MHTML) and [Webarchive](https://en.wikipedia.org/wiki/Webarchive) 格式的文档。
- 对保存的页面进行高亮、批注。
- 创建自定义筛选，过滤出需要的页面。
- 通过 RSS 订阅输出筛选后页面。
- 不依赖服务，所有功能都是离线的。
- 全文搜索
- 开源 SDK
- 暗黑模式
